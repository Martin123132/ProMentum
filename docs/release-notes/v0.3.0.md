# ProMentum v0.3.0 Release Review

ProMentum v0.3.0 turns the app into a fuller local-first project companion.
The main change is the new **Today** page: open the app, get one recommended
next action from a saved project, give it ten honest minutes, then log a win or
blocker.

## Added

- Added `/today` as the new home page.
- Added a local Today recommendation API at `GET /api/today`.
- Added project stages: `Spark`, `Shaping`, `First Step`, `In Progress`, `Parked`, and `Done`.
- Added project wins, blockers, and next-action history.
- Added Today flow actions for starting a 10-minute pass, marking an action done, parking a blocked project, and logging wins/blockers.
- Added export formats for Project Brief, Today Plan, and Progress Log.

## Still Local First

- No API keys.
- No cloud calls.
- No accounts.
- No npm or build step.
- Runtime data stays under `PROMENTUM_HOME`, defaulting to `D:\ProMentumData` on Windows when available.

## Manual Smoke Loop

1. Launch from `START_ProMentum_WINDOWS.bat`.
2. Confirm the app opens on Today.
3. Generate a first move if no project exists.
4. Save the result as a Momentum project.
5. Add or confirm one tiny action.
6. Return to Today and start the 10-minute flow.
7. Mark the action done or log a blocker.
8. Export Today Plan and Progress Log.
9. Reopen the app and confirm the project history is still present.
