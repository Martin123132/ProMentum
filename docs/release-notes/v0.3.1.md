# ProMentum v0.3.1 Release Review

ProMentum v0.3.1 is a first-run friction patch after testing the public
v0.3.0 ZIP. No v0.4.0 features were added.

## Fixed

- Fixed the Windows launcher falling back to the portable `promentum_data`
  folder when `D:\ProMentumData` already existed.
- Kept the default Windows storage path on D: when available.

## Added

- Added `docs\TESTER_CHECKLIST.md` for public tester runs.
- Added `docs\FEEDBACK_TEMPLATE.md` for plain feedback collection.

## Public v0.3.0 Tester Pass Finding

The public `ProMentum-v0.3.0.zip` downloaded and verified successfully.
Launching from the extracted ZIP worked, but the launcher reported:

```text
Data folder: <unzipped folder>\promentum_data
```

That happened because `md D:\ProMentumData` can return an error when the folder
already exists. v0.3.1 now creates the folder only when it is missing, then
falls back to portable storage only if the D-drive folder still cannot be used.
