# ProMentum Tester Checklist

Use this when asking a real person to try ProMentum from the public ZIP.
The goal is not feature feedback yet. The goal is proving first-run friction.

## Setup

1. Download the latest `ProMentum-vX.Y.Z.zip` from GitHub Releases.
2. Unzip it somewhere normal, such as Desktop or D: drive.
3. Double-click `START_ProMentum_WINDOWS.bat`.
4. If Windows asks about Python, write down exactly what happened.
5. If the browser opens, start the timer.

## First Run

1. Confirm the first page is **Today**.
2. Press **Generate First Move** if Today has no project.
3. Save the result as a Momentum project.
4. Confirm Momentum shows a project stage and traffic light.
5. Return to Today.
6. Press **Start 10 Minutes**.
7. Mark one action done, or log one blocker.
8. Export **Today Plan** and **Progress Log**.
9. Use **Open Exports Folder** if it appears.
10. Close the app window and browser tab.

## What To Record

- Time from unzip to first generated result.
- Whether Python blocked the tester.
- Whether Windows security prompts caused confusion.
- Whether the launcher text made sense.
- Whether Today explained what to do next.
- Whether Save Project made sense.
- Whether exports were easy to find.
- The funniest or most useful generated seed/mode.

## Pass Condition

A tester passes if they can unzip, launch, generate, save a project, use Today,
and export a file in under 10 minutes without needing technical help.
